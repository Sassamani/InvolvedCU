# InvolvedCU

InvolvedCU is a university-oriented web application designed to help students discover clubs, explore events, and become more engaged in campus life. The platform centralizes information about student organizations and activities, making it easier for users to find opportunities that match their interests.

The project focuses on usability, accessibility, and user-centered design principles. It was developed as part of an academic course and follows a structured design process including research, prototyping, and evaluation.

---

## Key Features

- Club discovery interface with detailed club pages  
- Events explorer with event details and descriptions  
- User profile and personalization concepts  
- Clean and intuitive UI designed using Figma  
- Focus on learnability and ease of navigation

## Project Context

This project was developed for an academic course in User Interface Design. The goal was to apply theoretical concepts such as user-centered design, usability evaluation, and iterative prototyping to a realistic university application.
